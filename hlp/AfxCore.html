#$KIntroduction

 

NetChess version 1.7 is a Graphical User Interface (GUI) for playing chess game. This software can be used

  1. To play chess between two players over network using TCP/IP (LAN) or in internet/intranet
  2. To send instant message(chat) while playing chess
  3. To observe the chess game played between two players over network
  4. To play chess between two players on the same board
  5. As PGN, FEN, EPD converter
  6. As PGN, FEN, EPD viewer
  7. To play chess using chess engines (Supports engines with WinBoard 1,2 and UCI 1,2 protocols)
  8. To play with chess engines on the same board or over network
  9. To play chess via email

 

 

#$KInstallation

 

To unzip Netchess.zip, WinZip software must be installed in the system.

 

  1. Download Netchess.zip file
  2. Extract all the files in Netchess.zip file. It will create NetChess directory in the extracted directory.
  3. The NetChess directory contains NetChess1.7.exe and NetChess1.7.hlp files.
  4. Play chess by clicking on NetChess1.7.exe

#$How to establish a chess connection in a network

 

To play chess in a network, you and your opponent must in the network. Before starting the game over network, If you are running as Chess server, give IP address and port number to your opponent. If you are running as client, ask IP address and port number from your opponent. Once IP address is known, use “ping <ipaddress>” to check the network connection with the server machine. If the other machine is able to ping, then NetChess can be used to play chess over network. If a firewall exists between two machines, then the port must be opened at firewall. Otherwise, NetChess will not work. Ask your network administrator for necessary help. Following steps describes the procedure to establish a chess connection in a network

 

  1. Run NetChess on two machines
  2. Run as Server in one machine and as Client in another machine (See steps 3 and 4)
  3. To run as Server, click Tools->Server or Server toolbar button and enter your name and port number (Default port number is 55555). Inform Server machine IP address and port number to the other player running as Client. Wait for the connection. First connection is your opponent and observers can be connected from second connection onwards.
  4. To run as Client, click Tools->Client or Client toolbar button and enter your name, server IP Address and Port number (if you don't know Server IP address, ask IP address and port number from the player running as Server)
  5. Server side player will decide Piece side or can opt for toss (You can discuss and decide who will play with White piece side). Once server accepts the piece side, connection will be established and piece side information will be sent to the client.
  6. Once the connection is established, click on “Edit->Manual Edit->Start Editing” to enable editing. Mutual agreement is required to enable Manual Editing feature. This step is required to enable the clock. Once the clock is enabled, it is advisable to disable manual editing feature. This will prevent unwanted undo, redo, movefirst, movelast, setting pieces etc... operations.
  7. To start clock, in Manual edit mode, click Edit ->Manual Edit -> Pause game (uncheck). This will also enables clock on the other player chess board.
  8. Status bar shows "Game started!" and clock start working. Now start playing the game
  9. Once the connection is established, use Tools->Send Message to send instant message or to Chat.
  10. White side player starts the game

 

#$How to connect as an Observer:

 

When two players are playing the chess over network and you would like to watch and chat then connect as an Observer. You can be observer only when two players have established a chess connection. You will be observer from second connection onwards to the chess server. Following steps describes the procedure to connect as an observer

 

  1. Prerequisite is Server and client must have established chess connection
  2. Know server IP address and port number (Ask for IP address and port number from server side player)
  3. Connect to Server by clicking Tools->Client and enter Name, Server IP address and port number
  4. Server may accept or reject your request
  5. During the Observer mode, you can only view or chat with others. Message will be broadcast to Server, client and all other observers. You can also do save and copy operations.
  6. Connection and disconnection can be done any time during the play

 

#$KHow to use as PGN/FEN/EPD viewer:

 

Manual editing feature should be enabled to view the chess games. NetChess can be used to view a single game or multiple games. This supports PGN, FEN, EPD positions or games.

 

Single game view:

  1. Open the game using File->Open or Copy and paste the game
  2. Use MoveFirst, MoveLast, Undo, Redo(Edit->Manual Editing->… or arrow marks on the tool bar) to navigate through the game
  3. If FEN/EPD positions are not proper order, use “Multiple gave view”

 

Multiple game views:

  1. Load the multiple games using File->Load games
  2. Navigate through the multiple games using LoadFirstGame, LoadPrevGame, LoadNextGame, LoadLastGame , Reload Game options in File commands or use <<, <, >, >> buttons in toolbar
  3. Once a single game is loaded, use move navigation(single game view) to navigate through the moves

 

Automatic view (Replay):

 

  1. Open the game using File->Open or Copy and paste the game
  2. Use View ->Replay ->Start to start Replay the game. Manual editing should be disabled to replay the game
  3. Game interval is based on configuration parameter in Edit->Preferences

 

#$KHow to use NetChess as PGN/FEN/EPD Converter

 

  1. Load the game using File->Load games
  2. Use File->Convert to convert between FEN, EPD and PGN formats
  3. Select the file, and click on convert. Each game will be converted and saved into <filename><sequence number>.<selected format>
  4. Note: As this is file operation, it will take several minutes to complete the task and completion time will be based on number of games loaded

 

#$How to play with Chess engine as White pieces side player:

 

  1. Select White Engine -> White as Engine (Observe a check to the menu item). On selecting White as Engine menu item, Default, Load and Engine Console menu items will be enabled
  2. If default game is already selected in Edit->Preferences, and want to start the default engine, select White Engine -> Default menu item
  3. If you want to load a new engine, click White Engine -> Load and select the engine executable from File dialog. Once the engine is loaded,  Reload, Start and Engine console menu items will be enabled
  4. Once the game is loaded, click Start to start the engine. Select the Engine protocol (WinBoard I, WinBoard II, UCI I, UCI II). Note the features/options supported by the engine. If required, send the options for UCI engines
  5. Click “Move Now” to start the engine to play the first move
  6. White Engine ->Engine console can be used to view engine interactions and to send additional commands to engine
  7. To close/quit the engine, click White Engine -> Close. This will quit the engine. To play again, Load the engine again
  8. Important Note: Before exiting NetChess, close chess engines. Else Chess engines will not be stopped, will be running in the background and also consumes most of your CPU time. In this case, use Task Manager to end chess engine process.

 

#$How to play with Chess engine as Black pieces side player

 

  1. Select Black Engine -> Black as Engine (Observe a check to the menu item). On selecting Black as Engine menu item, Default, Load and Engine Console menu items will be enabled
  2. If default game is already selected in Edit->Preferences, and want to start the default engine, select Black Engine -> Default menu item
  3. If you want to load a new engine, click Black Engine -> Load and select the engine executable from File dialog. Once the engine is loaded,  Reload, Start and Engine console menu items will be enabled
  4. Once the game is loaded, click Start to start the engine. Select the Engine protocol (WinBoard I, WinBoard II, UCI I, UCI II). Note the features/options supported by the engine. If required, send the options for UCI engines
  5. If you are playing with White piece side, then move now
  6. Black Engine ->Engine console can be used to view engine interactions and to send additional commands to engine
  7. To close/quit the engine, click Black Engine -> Close. This will quit the engine. To play again, Load the engine again
  8. Important Note: Before exiting NetChess, close chess engines. Else Chess engines will not be stopped, will be running in the background and also consumes most of your CPU time. In this case, use Task Manager to end chess engine process.

 

#$How to play White and Black with Check engines on the same board

 

  1. Follow the steps mentioned in How to play with Chess engine as White pieces side playerwhite_engine till step 4
  2. Follow the steps mentioned in How to play with Chess engine as Black pieces side playerblack_engine till step 4
  3. Once two engines are loaded and started, Client White Engine -> Move Now
  4. Engine console can be used to view engine interactions and to send additional commands to engine
  5. To close/quit the engine, click Black/White Engine -> Close. This will quit the engine. To play again, Load the engine again
  6. Important Note: Before exiting NetChess, close chess engines. Else Chess engines will not be stopped, will be running in the background and also consumes most of your CPU time. In this case, use Task Manager to end chess engine process.

 

 

#How to play White with chess engine on One board and Black chess on other board in Network mode

 

  1. Establish a network connection as mentioned in How to establish a connection in a networknetwork_connection
  2. Configure White Engine in first board as mentioned in How to play with Chess engine as White pieces side playerwhite_engine
  3. Configure Black Engine in second board as mentioned in How to play with Chess engine as Black pieces side playerblack_engine
  4. Click White Engine-> Move now in the first board(White piece side board)

 

Note: NetChess can also be used to play chess with chess engine in one machine and without engine (as Human player) in other machine

 

#How To play chess via email:

 

  1. Start chess board and move now
  2. Click Edit->Copy to copy as PGN
  3. Send the PGN data as mail (Your friend receives the PGN file, follow step 4 to play)
  4. (Other player) Receive the PGN file as mail
  5. Copy and paste the PGN file in NetChess
  6. Move now
  7. Repeat step 1) to 6)

 

#$KHow to configure chess engine

 

  1. Chess engine can be configured as either White side player or black side player
  2. Each chess engine will have a binary file available in the chess engine directory
  3. Enable the White as Engine for White side player and Black as Engine for Black side player
  4. You can set default check by configuring the chess engine in Edit->Preferences
  5. Once the Chess engine menu is enabled and want to play with default chess engine Click White Engine -> Default(for white side) or Black Engine ->Default(for black side)
  6. For loading a chess engine without default chess engine, White/Black Engine->Load and select the Chess engine binary
  7. Select the Engine protocol  by clicking White/Black Engine -> Start
  8. To move click White Engine -> Move now
  9. Engine console can be used to view engine interactions and send additional commands
  10. Once the game is complete don’t forget to close the engine(Engine->Close)

#$KHow to save and restart the game over network or on the same board

 

  1. Save the game as PGN format(File->Save will save the game in PGN format)
  2. If you would like to play chess over network with the saved game, establish a network connection as mentioned in the section How to establish a connection in a networknetwork_connection
  3. Once the connection is established, Fille->Open or Copy and Page the game. This can be done by any player
  4. If the game is on the same board continue from step 5
  5. Decide who has to continue the chess and Change the players turn using Edit->Manual Editing -> Player’s Trun
  6. Start playing the chess
  7. Chess engines also configured accordingly

$K#New game

 

Starts new game. If the game is in progress, Prompts for saving the current playing game in PGN format. On Network game mode, Sends a new game message to the opponent client, displays a message “Your opponent has requested for new game” and Prompts for saving the current playing game. If engine is playing, sends new game command based on WinBoard or UCI protocol.

$K#Open

 

Open PGN/FEN/EPD games/positions. Displays Open Dialog. On selecting a file and click on Open in the Open dialog, Opens the game based on file extension. Allowed file extensions are .fen, .epd, .pgn. $K#Preferences

 

This displays configuration parameter dialog box. The configuration parameters will be loaded from NetChess.ini file. The dialog box contains following configuration parameters:

  • White Color – Select White piece side color
  • Black Color – Select Black piece side color
  • Auto Save – Auto save option. When the option is checked, game will save for every 5 seconds. Select file name from File->Save(first time)
  • Save with comments – Save the PGN/EPD with comments
  • Log NetChess actions – Log NetChess actions, view the NetChess log from View-Log
  • PGN mail format – Save the moves with spaces between move number and move. Example 1.Nf3 g4 2.a6 Ng2 etc..
  • UCIMove option->Start Position – Send the “start position” command to UCI engines
  • UCIMove option->FEN string – Send the “start FEN” command to UCI engines
  • Default white Engine – Default engine file name and path for white side pieces. Select the engine from “Browse” button
  • Default black Engine – Default engine file name and path for black side pieces. Select the engine from “Browse” button
  • Open game interval(in seconds) – Interval between two moves while Displaying the game
  • OK button – Store above configuration parameters in the current application only
  • Apply/Save button – Store above configuration parameters in NetChess.ini
  • Cancel button – Do nothing
  • Select default piece set or Font based piece set

$K#Start Editing

 

This Enables/disables Editing feature. While on network mode, sends a request for confirmation to the opponent. In the Edit mode, following actions can be done

  • Undo – Undo the previous move
  • Redo – Redo the previously undone move
  • MoveFirst – Move to First position of the game

#$KTroubleshooting

 

1)       NetChess not responding

This may be due the Chess engine which is loaded through NetChess is not responding to NetChess. Please close engine and NetChess using Task manager.

 

2)       After NetChess is exit, CPU utilization is very high

This is due to Chess engine which is still running in the background. Use Task Manager to kill the chess engine

 

3)       While playing NetChess using Chess engines, CPU utilization is very high

This may be due to chess engine. Use the Task Manager to view the CPU utilization.
